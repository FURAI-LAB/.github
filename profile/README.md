## SEO & Performance
We treat search visibility and load performance as part of the architecture, not an afterthought.

**Technical SEO** тАФ canonical URLs, hreflang, structured data (JSON-LD), dynamic sitemaps, and robots.txt built into the routing layer.

**Core Web Vitals** тАФ image pipelines optimized for LCP, font loading tuned to avoid layout shift, SSR with no client-side wait.

**Multilingual by design** тАФ separate localized routes with proper hreflang annotation, not client-side string-swapping that search engines can't index.

**Search Console & Webmaster** тАФ verified, monitored, and submitted from day one. Google Search Console and Yandex Webmaster both wired in.
